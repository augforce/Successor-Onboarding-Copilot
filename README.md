# 🤖 Successor Onboarding Copilot
**Role:** IT Operations Transition Assistant  
**Stack:** Microsoft Azure AI Foundry | GPT-4.1-mini | Azure AI Search (RAG)

## 🎯 Project Overview
I developed this Custom Copilot to solve the "Knowledge Gap" during employee transitions. 
The agent is grounded in specific IT Standard Operating Procedures (SOPs) to answer 
questions for my successor regarding macOS onboarding, Google Workspace security, risk assessments, etc.

## 🛠️ Technical Implementation
- **Model:** GPT-4.1-mini (chosen for high reasoning at 90% lower cost).
- **RAG Architecture:** Integrated Azure AI Search with a manual Vector Store for document retrieval.
- **Prompt Engineering:** Designed a system persona to act as a mentor, citing specific documentation for every answer.
